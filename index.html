Como usar o Kinect junto ao Raspberry










Kinect

O Kinect é um sensor de movimentos que foi desenvolvido com intuito de melhorar a experiência dos jogadores e a jogabilidade, devido ao fato de dispensar o uso de controles ou joysticks para jogar.

A tecnologia do Kinect permite que os jogadores interajam livremente com os jogos de videogame, sem ter a necessidade de ter o controle em mãos, ou seja, apenas através dos movimentos do próprio corpo. Este dispositivo foi considerado uma inovação, principalmente pelas tecnologias já utilizadas pelos concorrentes.

O Kinect funciona através de duas câmeras, uma que reconhece o rosto da pessoa e outra infravermelha, que é capaz de reconhecer os movimento e profundidades. Esta última câmera divide o corpo humano em 48 pontos, que são identificados em tempo real, através de sensores.

O aparelho ainda é equipado com um microfone micro-vetorial, capaz de captar e isolar a fala dos jogadores do som ambiente, além de conseguir captar comandos de voz.

O Kinect compreende linguagens, sinais e movimentos, e faz com que os jogadores possam jogar e brincar, sem precisar estar segurando nada para que o jogo faça os mesmos movimentos.

Raspberry

O Raspberry Pi é um computador de baixo custo e bem pequeno aproximando-se muito ao tamanho de um cartão de crédito, foi desenvolvido no Reino Unido pela Fundação Raspberry Pi. Para usá-lo, basta plugar um teclado e um mouse padrão a ele e conectar tudo isso a um monitor ou a uma televisão. Os modelos custam entre US$ 25 e US$ 35.

A função básica desse equipamento é oferecer uma alternativa barata, prática e acessível para que pessoas de várias idades possam explorar todas as capacidades da computação. Além disso, também visa facilitar a aprendizagem de programação em linguagens como Scratch e Python.

Algumas utilidades do dia a dia incluem criação de servidor de arquivos pessoal, videogame retrô e outras.

Como usar?

Para começar a utilização do Raspberry é necessário roda um sistema operacional como Ubuntu, Raspbian e outras distribuições do Linux, nesse tutorial será usado o Raspbian, após realizar a instalação ligamos o Kinect e conectamos a entrada usb no Raspberry.

E então podemos fazer uso do Kinect com o Raspberry de diversas maneiras, mas nesse tutorial será mostrado apenas duas formas de utiliza-lo, à primeira será como uma simples câmera e a segunda como um dispositivo que possamos controlar com programação.

  • Como uma câmera
  • Feramenta para programação

Já que é desejado usar apenas vídeo existe um driver no kernel! Que pode ser carregado assim:

modprobe videodev
modprobe gspca_main
modprobe gspca_kinect

Ao fazer isso você obtém um novo / dev / videoX e pode usá-lo como uma webcam, mas para testar se está conectado com a câmera do Kinect ao Raspberry, instale o guvcview.

sudo apt-get install guvcview

Em seguida, digite sudo guvcview no terminal e ele deverá abrir um painel de opções e a visão de controle da câmera, mas se retorna algum erro pode ser o audio então execute esse comando sudo guvcview --audio=none, e se tudo ocorreu bem abrirá algo parecido com essa imagem.

Principais ferramentas para o tutorial

Raspbian
Python3
OpenCV 3.4
Libfreenect

Expandir o sistema de arquivos
Digite o seguinte comando para expandir o sistema de arquivos Raspberry Pi3
sudo raspi-config
Em seguida, selecione o seguinte: Opções avançadas >> A1 Expandir sistema de arquivos >> Pressione "Enter" e será exibido uma mensagem "A partição raiz foi redimensionada", Então reinici o sistema.

Instalando as dependências
A primeira etapa é atualizar e atualizar todos os pacotes existentes e enseguida reinici

sudo apt-get update
sudo apt-get upgrade
sudo shutdown -r now

Após reiniciar, inicie o Terminal novamente. Faça o seguinte.

Instale os pacotes de desenvolvedor do CMAKE
sudo apt-get install build-essential cmake pkg-config -y

Instalar pacotes de E / S de imagem
sudo apt-get install libjpeg-dev libtiff5-dev libjasper-dev libpng12-dev -y

Instalar pacotes de E / S de vídeo
sudo apt-get install libavcodec-dev libavformat-dev libswscale-dev libv4l-dev -y
sudo apt-get install libxvidcore-dev libx264-dev -y

Instale a biblioteca de desenvolvimento GTK para janelas básicas da GUI
sudo apt-get install libgtk2.0-dev libgtk-3-dev -y

Instale pacotes de otimização (operações de matriz aprimoradas para OpenCV)
sudo apt-get install libatlas-base-dev gfortran -y

Instale o Python 3, setuptools, dev e Numpy

sudo apt-get install python3 python3-setuptools python3-dev -y
wget https://bootstrap.pypa.io/get-pip.py
sudo python3 get-pip.py
sudo pip3 install numpy

Faça o download dos módulos extras OpenCV 3.4 e contrib

cd ~
wget -O opencv.zip https://github.com/Itseez/opencv/archive/3.4.0.zip
unzip opencv.zip
wget -O opencv_contrib.zip https://github.com/Itseez/opencv_contrib/archive/3.4.0.zip
unzip opencv_contrib.zip

Compile e instale o OpenCV 3.4.0 para Python 3 
  cd opencv-3.4.0
  mkdir build
  cd build
  cmake -D CMAKE_BUILD_TYPE=RELEASE \
  -D CMAKE_INSTALL_PREFIX=/usr/local \
  -D BUILD_opencv_java=OFF \
  -D BUILD_opencv_python2=OFF \
  -D BUILD_opencv_python3=ON \
  -D PYTHON_DEFAULT_EXECUTABLE=$(which python3) \
  -D INSTALL_C_EXAMPLES=OFF \
  -D INSTALL_PYTHON_EXAMPLES=ON \
  -D BUILD_EXAMPLES=ON\
  -D OPENCV_EXTRA_MODULES_PATH=~/opencv_contrib-3.4.0/modules \
  -D WITH_CUDA=OFF \
  -D BUILD_TESTS=OFF \
  -D BUILD_PERF_TESTS= OFF ..

Digite o seguinte comando para compilá-lo usando 4 núcleos de pi make -j4

Após a compilação bem-sucedida, instale a compilação usando o seguinte comando sudo make install && ldconfig

Libfreenect e suas dependências
sudo apt-get install git-core cmake freeglut3-dev pkg-config build-essential libxmu-dev libxi-dev libusb-1.0-0-dev

Clone o repositório libfreenect no seu sistema
git clone https://github.com/OpenKinect/libfreenect.git

Instale a libfreenect 

  cd libfreenect
  mkdir build
  cd build
  cmake -L ..
  make
  sudo make install
  sudo ldconfig /usr/local/lib64/
Para testar execute: sudo freenect-glview

Para usar o Kinect com opencv e python, é necessário instalar os wrappers python para libfreenct e as dependências necessárias 

  sudo apt-get install cython
  sudo apt-get install python-dev
  sudo apt-get install python-numpy
Depois vá para o diretório /Libfreenect / wrappers / python e execute o seguinte comando sudo python setup.py install
